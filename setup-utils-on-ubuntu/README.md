## 3. ユーティリティツールのインストール

### Java コマンド (JDK)

Java 実行環境の JDK (Java Development Kit) をインストールします。 GUI 機能を含まない軽量な ```headless``` 版を使用します。

なお、各 JDK のインストール手順の最後にあるバージョン確認で、意図しないバージョンになっている場合は、「[特定のバージョンに切り替える](#特定のバージョンに切り替える)」章の手順を実施してください。

#### JDK 21

1. インストールを実行します。
	```
	$ sudo apt install -y openjdk-21-jdk-headless
	```
1. インストールされたことを確認します。バージョン情報が表示されれば成功です。
	```
	$ java -version
	```

#### 特定のバージョンに切り替える

一つの端末に Java 11, Java 17, Java 21 と複数のバージョンがインストールされている場合、Linux の仕組み（alternatives）によって「デフォルトで使用されるバージョン」が管理されています。 意図したバージョンへ切り替える手順は以下の通りです。

1. 現在「Java 17」がアクティブになっており、「Java 21」に切り替えたい状況と仮定します。
	```
	$ java -version

    openjdk version "17.0.18" 2026-01-20
	```

1. Java を対象としてバージョン管理プログラムを起動します。
	```
	$ sudo update-alternatives --config java
	```

1. リストの先頭で「*」が付いている「Java 17」が現在アクティブな状態であることが分かります。
	```
	alternative java (/usr/bin/java を提供) には 3 個の選択肢があります。

	  選択肢    パス                                       優先度  状態
	------------------------------------------------------------
	  0            /usr/lib/jvm/java-21-openjdk-amd64/bin/java   2111      自動モード
	  1            /usr/lib/jvm/java-11-openjdk-amd64/bin/java   1111      手動モード
	* 2            /usr/lib/jvm/java-17-openjdk-amd64/bin/java   1711      手動モード
	  3            /usr/lib/jvm/java-21-openjdk-amd64/bin/java   2111      手動モード

	現在の選択 [*] を保持するには <Enter>、さもなければ選択肢の番号のキーを押してください:
	```

1. プロンプトに対して、使いたい「Java 21」に該当するリスト番号の「3」を入力します。
	```
	現在の選択 [*] を保持するには <Enter>、さもなければ選択肢の番号のキーを押してください: 3

	update-alternatives: /usr/bin/java (java) を提供するためにマニュアルモードで /usr/lib/jvm/java-21-openjdk-amd64/bin/java を使います
	```

1. バージョン管理プログラムが終了するので、再度バージョンを確認すると「Java 21」に切り替わったことが確認できます。
	```
	$ java -version

	openjdk version "21.0.10" 2026-01-20
	```

### jq コマンド

JSON 形式のデータを整形したり、特定の値を抽出・加工したりするために使用します。

1. インストールを実行します。
	```
	$ sudo apt install -y jq
	```
1. インストールされたことを確認します。バージョン情報が表示されれば成功です。
	```
	$ jq -V
	```

### unzip コマンド

Zip 形式の圧縮ファイルを展開するために使用します。

1. インストールを実行します。
	```
	$ sudo apt install -y unzip
	```
1. インストールされたことを確認します。バージョン情報が表示されれば成功です。
	```
	$ unzip -v
	```
