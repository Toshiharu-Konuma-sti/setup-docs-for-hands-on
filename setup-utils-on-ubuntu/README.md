## 3. ユーティリティツールのインストール

### Java コマンド (JDK)

Java 形式のプログラムを実行・開発するために使用します。

1. インストールを実行します。
	```
	$ sudo apt install -y openjdk-21-jdk-headless
	```
2. インストールされたことを確認します。バージョン情報が表示されれば成功です。
	```
	$ java -version
	```

### jq コマンド

JSON 形式のデータを整形したり、特定の値を抽出・加工したりするために使用します。

1. インストールを実行します。
	```
	$ sudo apt install -y jq
	```
2. インストールされたことを確認します。バージョン情報が表示されれば成功です。
	```
	$ jq -V
	```

### unzip コマンド

Zip 形式の圧縮ファイルを展開するために使用します。

1. インストールを実行します。
	```
	$ sudo apt install -y unzip
	```
2. インストールされたことを確認します。バージョン情報が表示されれば成功です。
	```
	$ unzip -v
	```
